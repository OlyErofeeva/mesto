# Место

Сервис Mesto - это интерактивная страница, где скоро можно будет публиковать фотографии, удалять их и ставить лайки.

https://olyerofeeva.github.io/mesto/

### Используемые технологии: HTML, CSS, JavaScript.
* отзывчивая вёрстка
* grid layout
* flexbox
* БЭМ
